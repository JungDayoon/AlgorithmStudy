# [BOJ 11052] 카드 구매하기 - Python

### :computer: Algorithm

> Dynamic Programming



### :computer: Logic

N개의 카드를 구매하는 최대값은 

(0개의 카드를 사는 경우의 최대값 + N개짜리 카드팩 금액), (1개의 카드를 사는 경우의 최대값 + (N-1)개짜리 카드팩 금액), ..., (N-1개의 카드를 사는 경우의 최대값 + 1개짜리 카드팩 금액)의 최대값으로 구할 수 있다.
