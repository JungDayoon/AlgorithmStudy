이분탐색, DP

dp배열에는 현재 탐색하는 값 중 작은 것이 들어간다.

bs(cmp)함수에서는 cmp값이 dp배열에서 위치해야하는 인덱스값을 반환해준다.

메인함수에서는 bs에서 반환한 인덱스가 dp배열보다 큰 경우: append 작은 경우: 해당 인덱스 값을 변경

파이썬에서는 이분탐색 라이브러리를 지원한다. - `from bisect import bisect_left`
