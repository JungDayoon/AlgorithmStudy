## *- DP -*

* **DP** - Bottom up 사용

* `int[][] dp = new int[N+1][3];`

  * `dp[i][0]` : `i`줄에 아무 사자도 배치하지 않는 경우의 수

  * `dp[i][1]` : `i`줄의 왼쪽 칸에 사자를 배치하는 경우의 수

  * `dp[i][2]` : `i`줄의 오른쪽 칸에 사자를 배치하는 경우의 수

  * 예) `N = 1`일 때,

    `dp[1][0] = 1`

    `dp[1][1] = 1`

    `dp[1][2] = 1`

* `int Bottom_up(int N, int[][] dp)`

  * `dp[i][0] = (dp[i-1][0] + dp[i-1][1] + dp[i-1][2]) % 9901`

    : `i`줄에 아무 사자도 배치하지 않는 경우의 수

    ​	= `i-1`줄에 아무 사자도 배치하지 않는 경우, `i-1`줄의 왼쪽 칸에 사자를 배치하는 경우, `i-1`줄의 오른쪽 칸에 사자를 배치하는 경우 모두 가능하다. 세 가지 값 더한 값이 `dp[i][0]`

  * `dp[i][1] = (dp[i-1][0] + dp[i-1][2]) % 9901`

    : `i`줄의 왼쪽 칸에 사자를 배치하는 경우의 수

    ​	= `i-1`줄에 아무 사자도 배치하지 않는 경우, `i-1`줄에 오른쪽 칸에 사자를 배치하는 경우. 두 가지 경우가 가능하다. 두 값을 더한 값이 `dp[i][1]`

  * `dp[i][2] = (dp[i-1][0] + dp[i-1][1]) % 9901`

    : `i`줄의 오른쪽 칸에 사자를 배치하는 경우의 수

    ​	= `i-1`줄에 아무 사자도 배치하지 않는 경우, `i-1`줄에 왼쪽 칸에 사자를 배치하는 경우. 두 가지 경우가 가능하다. 두 값을 더한 값이 `dp[i][2]`

  * `(dp[N][0] + dp[N][1] + dp[N][2]) % 9901`를 리턴한다.
