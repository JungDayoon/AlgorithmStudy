## 접근법

> 크기가 N*N인 행렬 A가 주어졌을때, A의 B제곱을 구하는 프로그램을 구하는 것.
- `square(arr, b)`함수에 행렬과, 거듭제곱 값인 b를 넘겨준다.

## 로직
- b가 1이면 return.
- b가 짝수이면 `거듭제곱한 행렬`과, `b/2`를 넘겨준다.
- b가 홀수이면  `거듭제곱한 행렬 * 현재 행렬`과 `b/2`를 넘겨준다.
