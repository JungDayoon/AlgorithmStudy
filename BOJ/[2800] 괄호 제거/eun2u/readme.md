## 분류💁

스택, 문자열

## 접근법

- 문자열을 -> `vector<int>` 로 바꾸어 저장한다.

- ex) (0/(0)) -> 1002021



## 시퀀스

1. 괄호쌍이 몇개인지 찾고, 괄호 순서에맞게 번호를 부여함. 숫자는 0임. 괄호는(1~ 최대 10)

2. 괄호쌍 몇개를 제거할지 조합을 통해 뽑음

3. 제거된 괄호쌍을 set에 insert함


## 틀렸습니다 이유

- 중복되는 문자열을 처리를 해주지 않았다. `set<int>` 을 사용하면 중복되는 문자열을 처리하며 정렬되며 저장된다.
