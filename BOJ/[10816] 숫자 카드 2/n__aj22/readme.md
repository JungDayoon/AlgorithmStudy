## Algorithm

## Description

`Dictionary` 를 사용했다.

처음 구현시, 넷째 줄에 있는 숫자가 겹칠 것이라는 생각을 못하고 구현해서 실패했다.

이를 고려하기 위해, 첫번 째 숫자 카드를 먼저 dictionary 를 만들고, 그 dictionary를 이용해 두번 째 있는 숫자의 개수를 찾았다.
