## 접근법

- 스티커 중에서 점수의 합이 최대이면서 두 변을 공유하지 않는 스티커 집합을 구해야 한다.
- `pic`은 그림에 있는 스티커 점수이고, `score`에 각 스티커 자리에서 최대로 가질 수 있는 점수를 저장한다.  
- 만약, 현재 스티커의 열이 i 라면, 가질 수 있는 최대 점수는 
    - score[0][i] = max(score[1][i-1] + pic[0][i], score[0][i-1]);
