## 놓친것
- 같은 파이프 방향으로 놓여져 있는곳에만 갈 수 있음
- 상->하 하->상 좌->우 우->좌 방향으로 연결되어 있어야 함
```python
(((tn+1)%2+(tn//2)*2) in tunnel[tunnelMap[newR][newC]])
```
0:상,1:하,2:좌,3:우<br>
0->1, 1->0, 2->3, 3->2 가 되는 수식이다. 각각이 다음 터널에 있으면이라는 뜻이다.<Br>
만약 현재 0(상)방향으로 가면 다음 터널에 1(하)방향으로 터널이 있어야 한다.<br>
