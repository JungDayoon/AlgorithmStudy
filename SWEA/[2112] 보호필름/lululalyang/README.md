## -*Backtracking*-

* `check_byK()`: 합격기준 통과 여부 확인

  * 같은 성분이면 `same++`을 해주고 `same`이 `K`이상이면 다음 column확인

    => 한 줄 통과하면 `f2 = 1`

  * 통과되지 않는 col있다면 바로 `return false`

  * 모든 col 통과했다면 `return true`

    

* `copyrow(int[] now, int[] result)`

  : `result`배열을 `now`배열 값으로 변경

  

* `chk_medi(int goal, int num, int prev)`  => ***Backtracking***

  * `goal`: 현재 약품처리 해야하는 row 수

  * `num`: 처리한 row 수

  * `prev`: 조합위한 index

  * `int[] arr` : 약품 처리 전 `film`의 성분 정보 저장할 배열

  * `prev`번째 줄 부터 `A`약품 처리 후 Backtracking (`chk_medi(goal, num+1, i+1)`)

    ​								`B`약품 처리 후 Backtracking(`chk_medi(goal, num+1, i+1)`)

  * 매 약품 처리후 성능검사 통과했으면(`Flag == 1`) `return`

    `A`, `B` 두 약품 모두 처리해도 성능검사 통과 못하면 `film`을 원래 성분으로 바꿔 줌.

* ***in `Main`***

  * 약품 처리 전 `film`이 합격기준 통과하거나, `K==1`이면 `result[t] = 0` (최소약품 투입 횟수: 0)

  * `for`문 돌면서 1줄부터 ~ `K-1`줄까지 약품처리 해주면서 성능검사

    ```java
    int l_k; //약품처리할 줄 수
    for(l_k=1; l_k<K; l_k++) {
        chk_medi(l_k, 0, 0); 
        if(Flag == 1) 
         	break;
    }
    ```

  * 약품처리 된 줄이 `K`보다 적을 때 성능검사가 통과됐다면 `result[t] = l_k`

  * `K-1`개의 줄을 약품처리해도 성능검사를 통과하지 못했다면 `result[t] = K`
