# Programmers 12971번 : 스티커 모으기(2)

## Algorithm

DP

## Description

1. i번 째 값은 (i-1번째 값), 혹은 (i-2번째 값 + 현재 값) 중 더 큰 값이다.
2. 0번 째 스티커를 떼면, 마지막 스티커는 떼지 못한다. 따라서 0번 째 스티커를 뗀 경우, 1번 째 스티커를 뗀 경우, 이 두 경우로 나눠 생각한다.
3. 두 경우 중 가장 큰 값이 정답이다.
