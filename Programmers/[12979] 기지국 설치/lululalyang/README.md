# [Programmers]/[12979] 기지국 설치

## *- Simulation -*

* `int w` : 전파의 도달 거리
* `int range = 1 + 2*w` : 기지국 하나당 전파 도달거리
  * 기지국이 있는 그 자리 `1` 
  *  양쪽으로 전파 `w`니까 `2*w`
* 기지국이 있는 위치에 따라 
  * 기지국을 설치해야하는 구간의 길이 / `range` 만큼 기지국 개수를 늘려준다
  * 이때 길이%`range` 가 `0`이 아니라면 +1해준다
