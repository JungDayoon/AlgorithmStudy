## Algorithm

Hash

## Description

**'solution2'** : 처음에 시도했던 방식
+ 옷의 종류가 n개라면, 옷을 1개부터 n개 까지 입을 수 있기 때문에, 각 경우를 모두 세어서 더해줬다.
+ 30개의 테스트 케이스 중 1개가 시간초과가 났다(테케 1)

**'solution'** : 참고해서 푼 방식
+ 간단한 공식으로 계산 가능하다
+ (각 옷의 종류+1의 곱)-1을 해주면 된다.
    + 마지막에 빼는 1의 의미는 아무 옷도 선택되지 않았을 경우이다.
